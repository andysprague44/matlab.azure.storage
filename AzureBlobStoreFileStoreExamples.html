Accessing Files in Azure Blob Store from MATLAB

Below are some examples of using the AzureGzipFileDataStore, a custom dataStore that can access azure blob store 'blobs' (specifically .csv.gz files).
For more details on using a datastore in MATLAB see https://www.mathworks.com/help/matlab/import_export/what-is-a-datastore.html
Before using any of the below functions, set an env variable in your command window to authenticate your env with Azure:
setenv MW_WASB_SECRET_KEY 'N21TNaN5VwEXDiIy6pfc0xhtsW5XJy0LzaNjR7zmTY6XGeTcO76QOqKr3zLnWhttAzGfEL6YFqw27mDxogeKTQ=='
The key is unique to each Azure storage account, and can be btained in the azure portal (portal.azure.com), Storage Account -> Access Keys. You only need to do this once. Please keep the secrets es for any PROD environements out of source code!
In the examples below we use scada countainer in ncxdatabricksdev01 storage account (for which the key above is correct)
Account = 'ncxdatabricksdev01';
